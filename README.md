# 📊 YouTube Channel Analytics: End-to-End Data Analysis Project

## 📌 Project Overview

This project presents an end-to-end data analytics solution on YouTube channel data. The objective was to extract meaningful insights from raw data using SQL, Python, Feature Engineering, and Power BI.

The project covers the complete analytics workflow, including data cleaning, exploratory data analysis (EDA), feature engineering, KPI generation, and dashboard creation for business intelligence and decision-making.

## 🎯 Business Problem

With thousands of YouTube channels generating billions of views and subscribers, it becomes difficult to identify growth patterns and top-performing channels.

This project aims to answer key business questions such as:

* Which channels have the highest audience reach?
* What factors contribute to channel growth?
* Which channels generate the most views per video?
* How do subscribers and views correlate?
* Which countries dominate the YouTube ecosystem?
